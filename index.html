Web teknolojileriyle ilgileniyorum. Arayüz, sistem ve güvenlik konularında sade ama sağlam işler üretmeye çalışıyorum.

Hakkımda

Web geliştirme sürecine sadece “site yapmak” olarak bakmıyorum. Yazdığım kodun okunabilir, sürdürülebilir ve mantıklı olmasına önem veriyorum.

Frontend tarafında modern arayüzler üretirken, arka planda sistemin nasıl çalıştığını da anlamaya çalışırım. Güvenlik benim için ekstra bir özellik değil, baştan düşünülmesi gereken bir gerekliliktir.

Odaklandığım Alanlar

Aşağıdaki konular üzerinde aktif olarak çalışıyorum ve kendimi bu alanlarda geliştirmeye devam ediyorum:

  • HTML, CSS ve JavaScript ile modern web arayüzleri
  • Cam efektli, animasyonlu ve sade tasarım yaklaşımları
  • JavaScript tabanlı küçük oyunlar ve etkileşimler
  • SSL, HTTPS yapılandırmaları ve domain yönetimi
  • Basit ama mantıklı sistem mimarileri

Canlı Proje

Teoriyle yetinmiyorum. Üzerinde çalıştığım projeleri canlı ortamda test etmeyi seviyorum.

Aşağıdaki bağlantı, geliştirdiğim JavaScript tabanlı oyun altyapısına gider. Kod, performans ve tarayıcı davranışlarını gerçek kullanıcı senaryolarında denemek için kullanıyorum.
